## To host:
Ensure Node.js is installed
Download `index.js`
Create a new Slack bot and copy the credentials 
Create a `.env` file and copy contents from the `example env`
Open a terminal
Proceed to the folder that your index.js and .env file are found in
Run ` npm install axios dotenv @slack/bolt `
Run `node index.js`
